# ❤️ 恋爱能力自测 - Love IQ Test

> **了解自己的不足，是提升恋爱能力的第一步。**

---

## 🎯 这个项目是做什么的？

很多人恋爱不顺，却不知道**问题出在哪里**。

这个项目的核心目标很简单：

✅ **帮助你了解自己在恋爱能力上存在的不足**  
✅ **提供有针对性的方向，让你知道该从哪里提升**  
✅ **用科学、结构化的方式，替代盲目试错**

这不是一个"心理测试玩一玩"的项目，而是一套**自我诊断和成长工具**。

---

## 🧠 测试涵盖哪些能力维度？

我们基于经典恋爱心理学理论，设计了 **20 道情景选择题**，覆盖以下 **5 个核心维度**：

| 维度 | 说明 | 为什么重要 |
|------|------|-----------|
| **自我价值感** | 你在关系中的底线、自尊和独立性 | 价值感低 = 容易讨好、被操控 |
| **情绪管理能力** | 面对冲突、拒绝、冷处理时的反应 | 情绪稳定 = 关系长期的基石 |
| **社交直觉与读人能力** | 识别对方兴趣指标（IOI）、意图和真实想法 | 读不懂人 = 容易自作多情或错过信号 |
| **关系推进能力** | 从认识到暧昧、从暧昧到确认关系的节奏掌控 | 推进力弱 = 容易卡在朋友区 |
| **冲突处理与边界设定** | 面对分歧时如何表达、协商和设立边界 | 边界模糊 = 关系失衡的根源 |

完成测试后，你会得到：
- ✅ **各维度得分**
- ✅ **能力雷达图**（直观看到强弱项）
- ✅ **个性化分析与提升建议**
